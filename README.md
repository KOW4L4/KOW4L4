## 💡 About The Project
This project is an interactive healthcare chatbot designed to provide initial symptom assessments. It leverages large language models and web speech technologies to create a seamless and accessible user experience for preliminary health checking.

## ✨ Key Features
* 🧠 **AI-Powered Analysis:** Integrates OpenAI GPT for natural language processing and accurate symptom evaluation.
* 🗣️ **Voice Interaction:** Supports real-time voice input and output via the Web Speech API.
* 🖼️ **Computer Vision Support:** Utilizes PIL for processing user-uploaded images for enhanced analysis.
* 📱 **Responsive UI:** Clean and interactive frontend built with HTML, CSS, JavaScript, and Bootstrap.

## 🛠️ Tech Stack
* **Backend:** Python, Flask
* **Frontend:** HTML, JavaScript, Bootstrap
* **Core APIs:** OpenAI API, Web Speech API
